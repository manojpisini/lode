## Overview

LODE is a developer productivity tool that helps you:
- **Scaffold** projects with templates, recipes, and snippets
- **Validate** naming conventions, secrets, and configuration
- **Orchestrate** AI agents with context compilation and planning
- **Monitor** projects with a daemon, TUI, and MCP server
- **Secure** code with secret scanning and redaction
- **Release** with structured versioning and changelogs
